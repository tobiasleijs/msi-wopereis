Smart connection

Tijdens de lessen van Smart connection heb ik mij bezig gehouden met het programmeren van een site en API’s. Zelf heb ik nog geen tot weinig programmeer ervaring, Dankzij de lessen en de hulp van Tobias heb ik een simpele site kunnen bouwen.

API's

Een API oftewel ‘Application programming interface’ is een interface die twee programma’s aan elkaar kan koppelen. Tijdens de minor heb ik twee API’s toegepast, een voor het weer in mijn woonplaats Aalten en een voor de NASA foto van de dag.

Web design skills
