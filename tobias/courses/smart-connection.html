API's

De data uit de weer API wordt gebruikt om het weer in een land of stad te tonen. Omdat het enkel tonen van deze data niet heel lastig is, heb ik voor iets meer uitdaging gekozen. Ik heb een invoerveld toegevoegd waarin je een stad of land kan invullen en zonder dat je op een knop hoeft te drukken, wordt het weer van de ingevoerde locatie getoond.

De Nasa API wordt gebruikt om de foto of video van de dag op te halen zodat deze vervolgens getoond kan worden. Voor het ophalen van data uit de API is een API-sleutel nodig, deze kun je aanvragen bij Nasa zelf. De groottes van de afbeeldingen en video's verschillen van dag tot dag. Daarom heb ik ervoor gekozen om via een JavaScript functie de grootte van de container waarin het weer staat weergegeven gelijk te maken aan de grootte van Nasa foto/video van de dag. Dit zorgt voor een mooie UI.

Website

Voor het vak Smart Connection hebben we leren werken met html, css en JavaScript. Persoonlijk had ik al een klein beetje ervaring met html en css, maar ik had nooit eerder gewerkt met JavaScript. Daarom heb ik mezelf bij dit vak proberen uit te dagen door een aantal animaties met JavaScript te maken. Zo kun je op mijn portfoliopagina kaarten met foto's van rechts naar links slepen. De positie van de foto op de kaart volgt de richting van je sleepbeweging. Dit zorgt voor een 3D of parallax effect.

Ik heb een pop-upvenster ontwikkeld dat wordt geopend door op een van de kaarten te klikken. In dit venster wordt de HTML-content van een specifiek vak weergegeven. Met behulp van een JavaScript-functie kan de content van de pop-up dynamisch worden geladen. Hierdoor is het mogelijk om de inhoud van verschillende vakken in één enkel pop-upvenster te tonen, zonder dat er voor elk vak een aparte HTML-pagina hoeft te worden aangemaakt. Deze aanpak past een belangrijk principe uit de programmeer- en webdevelopmentwereld toe: DRY (Don't Repeat Yourself).
