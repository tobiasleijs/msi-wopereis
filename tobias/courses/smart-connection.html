API's

De weer API wordt gebruikt om het weer in een land of stad te tonen. Omdat het het enkel tonen van wat API data niet heel lastig is, heb ik ervoor gekozen om het mezelf iets lastiger te maken. Ik heb een invoerveld toegevoegd waarin je een stad of land kan invullen en zonder dat je op een knop hoeft te drukken, wordt het weer van de ingevoerde locatie getoond.

De Nasa API wordt gebruikt om de foto of video van de dag op te halen zodat deze vervolgend getoond kan worden. Voor het ophalen van data uit de API is een API-sleutel nodig, deze kun je aanvragen bij Nasa zelf. Omdat de groottes van de afbeeldingen en video's per dag nogal kunnen verschillen, heb ik ervoor gekozen om grootte van de weer api gelijk te maken aan de grootte van de Nasa api. Dit zorgt voor een mooie UI.

Website

Voor het vak Smart Connection hebben we leren werken met html, css en JavaScript. Persoonlijk had ik al een klein beetje ervaring met html en css, maar ik had nooit eerder gewerkt met JavaScript. Daarom heb ik me zelf bij dit vak proberen uit de dagen door een aantal animaties met javascript te maken. Zo heb ik bijvoorbeeld een animatie gemaakt waarbij je op de home pagina van de website en van mijn portfolio de kaarten met foto's van links naar rechts kan slepen. Tegelijkertijd wordt de positie van de foto op de kaart ook met de sleep richting mee veranderd. Dit zorgt voor een 3D of parallax effect.

Daarnaast heb ik ook een modal gemaakt die je kan openen door op een kaart te klikken. In deze modal wordt de html content van een vak getoond. Dit zorgt ervoor dat ik niet voor elk vak een nieuwe pagina hoef te maken, maar dat ik de content van de vakken in een html bestand kan zetten en deze kan tonen in de modal. Dit zorgt ervoor dat een belangrijk principe van programmeren en dus ook webdevelopment wordt toegepast, namelijk DRY (Don't Repeat Yourself).
