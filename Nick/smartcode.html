Smart Code

Inleiding

Het vak Smart Technologie begon met het maken van een IoT-schakeling die de lichtwaardes en de vochtigheid van een plant zal reguleren. De eerste les was gelijk Studio Visual Code, iets waar ik nog geen ervaring mee had en vanaf een nulpunt moest aanleren. Dit is dan ook mijn eerste leerontwikkeling, programmeren met C++ (Embedded). Het koste veel moeite en ik heb mezelf vaak de vraag gesteld: “Hoe gaat ik dit nou voor mekaar krijgen”, maar uiteindelijk is het toch gelukt en heb ik na de eerste les een dimmend lichtje geprogrammeerd die aangaat wanneer de sensor minder tot geen licht meer kan scannen. Rechts van deze alinea is het resultaat te zien. Later heb ik nog andere sensoren geprogrammeerd en aangesloten op de ESP32 chip. Deze sensoren verzamelen data en in de tweede les heb ik geleerd hoe ik de verzamelde informatie uit de aangesloten sensoren kan uploaden naar de Cloud. Hier gebruikte we een Influx-databaseserver voor. Aan het einde van de les heb ik geleerd hoe ik een code aan kan maken die de verzamelde info kan uploaden naar de Cloud. Hiermee kan ik uiteindelijk het plantenbakje automatisch verzorgen. Deze dataserver kan gevisualiseerd worden als is een dashboard. Hoe dat werkt heb ik geleerd in de Smart Connection lessen en daar is dan ook mijn dashboard in te zien. Hier een hyperlink naar de pagina van het dashboard.

Printed circuit board

In de derde les heb ik geleerd hoe ik een printed circuit board kan maken. De app waarmee ik gewerkt heb is autodesk Eagle. In deze app heb ik geleerd hoe ik een elektrisch schema kan maken en daarna een design te ontwerpen met alle componenten erin. De stappen waren wederom lastig om te volgen aan het begin, maar uiteindelijk heb ik het onder de knie gekregen en is op de afbeelding links van deze alinea het resultaat te zien. Later heb ik deze PCB nog gesoldeerd en dat kun je lezen in de volgende alinea.

Solderen

Het PCB’tje werd besteld en toen deze binnenkwam mochten we deze zelf solderen. Ik heb weinig ervaring met solderen, ik heb het alleen een paar keer gedaan in groep 8 tijdens knutsellessen. Dus een kleine basis. In het filmpje rechts van deze alinea is een timelaps te zien van hoe ik en paar stukjes soldeer aan mijn pcb.

Conclusie
Tijdens de module Smart Technologie heb ik dus veel nieuwe vaardigheden aangeleerd. In de eerste les heb ik de basis van programmeren in Studio Visual Code onder de knie gekregen en hoe ik sensoren op de ESP32 kan aansluiten Dit gaf me een beter inzicht in hoe technologie werkt, ondanks dat ik hier geen voorkennis van had. In de tweede les leerde ik om gegevens van sensoren naar de Cloud te uploaden met behulp van de Influx-databaseserver. De derde les ging over het ontwerpen van een printed circuit board (PCB) in Autodesk Eagle, waarbij ik een elektrisch schema en een ontwerp voor een fysieke printplaat maakte. Dit heeft me duidelijk gemaakt hoe indrukwekkend zulke kleine chips eigenlijk wel niet zijn. Deze module heeft me niet alleen technische kennis opgeleverd, maar ook laten zien hoe belangrijk doorzettingsvermogen is bij het leren van nieuwe vaardigheden. Want al had ik dat niet gehad, dan was me dit allemaal ook niet gelukt
