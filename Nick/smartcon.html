Smart Connection

Inleiding

Tijdens de trainingen van Smart Connection heb ik nieuwe vaardigheden ontwikkeld op het gebied van programmeren, webdesign en API’s. Ik leerde Javascript, HTML en CSS, waarmee ik een website heb gebouwd. Mijn webdesignvaardigheden zijn ontwikkeld en deze verbetering is ook terug te zien op de website. Deze training combineert techniek, creativiteit en doorzettingsvermogen. En persoonlijk vond ik dit erg leuk om te leren, vooral webdesign.

Programmeren
Ik heb verschillende codetalen geleerd, namelijk Javascript, HTML en CSS. Javascript heb ik gebruikt voor de API’s en HTML en CSS heb ik gebruikt voor de website. Deze “talen” kende ik hiervoor nog niet en Javascript vind ik nog lastig, omdat ik enkel API’s ermee gemaakt heb. HTML en CSS kan ik wel beter mee overweg en deze talen heb ik geleerd.

Web design skills
Voor mijn web design skills wil ik graag verwijzen naar mijn website. Klik en kijk maar een beetje rond, want ik vind dat het aardig goed gelukt is voor iemand met geen achtergrond in SVC. De website word ook steeds mooier ontworpen en zo is mijn ontwikkeling te volgen. Dit was mijn eerste pagina en is vandaar nog niet zo mooi ontworpen als andere. Smart Technologie ben ik zelf erg trots op evenals de homepage en Smart Business.

API's
Voor mijn API's verwijs ik graagn naar mijn homepage. Daar heb ik knopjes gecreëerd die je aan kan klikken en die verwijzen je naar de website!
